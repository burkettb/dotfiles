# Marketing UI kit — Slate

A long-scroll marketing landing page for the Slate billing product. Composes the design-system primitives.

## Sections (`MarketingPage.jsx`)
- **SiteNav** — sticky blurred bar, links, sign-in + trial CTA.
- **Hero** — editorial asymmetric layout: oversized display headline + narrow copy column left, product visual (a floating invoice card) right. One gradient-free surface.
- **Features** — symmetric 4-up row, icons from one set, text left-aligned.
- **Pricing** — three tiered cards, one promoted (accent border), 4–6 bullets each + "compare all" link.
- **Testimonials** — quote grid; every quote makes a specific claim, no stars.
- **CTA** — dark solid band with one primary action.
- **SiteFooter** — link columns + placeholder legal line.

## Run
Open `index.html`. Requires the compiled `_ds_bundle.js` at the project root.

## Playbook applied
Editorial asymmetric hero (no centered stack) · long-scroll narrative (problem → product → proof → pricing) · tiered pricing with one promoted tier · specific-claim testimonials · house headline rule (states a concrete outcome, no "Supercharge/Reimagine" templates).
