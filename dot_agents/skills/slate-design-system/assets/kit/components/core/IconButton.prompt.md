# IconButton

A square, icon-only action for toolbars, table row actions, and dismiss controls. Always pass `label`.

```jsx
<IconButton icon="ellipsis-horizontal" label="Row actions" />
<IconButton icon="x-mark" label="Close" size="sm" />
<IconButton icon="pencil-square" label="Edit" outline />
```

Ghost by default; `outline` for standalone controls needing a boundary.
