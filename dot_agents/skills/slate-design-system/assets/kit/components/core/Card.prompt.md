# Card

A bordered content region — hairline border, no shadow at rest. Use `title`/`description` for the common header, or compose subcomponents.

```jsx
<Card title="Billing" description="Manage your plan and payment method">
  <p>Content…</p>
</Card>

<Card>
  <Card.Header>…</Card.Header>
  <Card.Body>…</Card.Body>
  <Card.Footer><Button variant="primary">Save</Button></Card.Footer>
</Card>
```
