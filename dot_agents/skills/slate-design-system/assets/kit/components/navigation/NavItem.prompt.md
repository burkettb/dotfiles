# NavItem

A sidebar navigation row. Active = 2–3px leading accent bar + stronger text. Renders as `<a>` with `href`, else `<button>`.

```jsx
<NavItem icon="home" active>Overview</NavItem>
<NavItem icon="document-text" href="/invoices">Invoices</NavItem>
<NavItem icon="cog-6-tooth">Settings</NavItem>
```
