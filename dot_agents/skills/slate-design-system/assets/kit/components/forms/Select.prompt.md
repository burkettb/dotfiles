# Select

Outlined native dropdown with a custom chevron. Pass `options` or `<option>` children.

```jsx
<Select label="Role" options={['Owner', 'Admin', 'Member']} />
<Select label="Plan" placeholder="Choose a plan"
  options={[{ value: 'pro', label: 'Pro' }, { value: 'team', label: 'Team' }]} />
```
