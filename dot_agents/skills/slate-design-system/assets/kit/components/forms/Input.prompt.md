# Input

Outlined text field. Label above, helper text or inline error below — the house form layout. Single column, one field per row.

```jsx
<Input label="Work email" type="email" placeholder="you@company.com" required />
<Input label="Team name" helpText="Shown on invoices." />
<Input label="Slug" error="This slug is already taken." defaultValue="acme" />
```
