# Skeleton

A loading placeholder — a quiet slate shimmer, never a spinner. Mirror the shape of the content that will replace it. Shimmer stops under `prefers-reduced-motion`.

```jsx
<Skeleton variant="circle" width={36} height={36} />
<Skeleton variant="line" width={180} />
<Skeleton variant="text" lines={3} />
<Skeleton variant="rect" height={120} radius={6} />
<Skeleton variant="button" />
```

Variants: `line | text | circle | rect | button`.
