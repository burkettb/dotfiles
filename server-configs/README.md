# Lightweight Server Configs

Minimal configurations for shared server environments.

## Quick Install

```bash
curl -sSL https://raw.githubusercontent.com/yourusername/server-configs/main/install.sh | bash
```

## Manual Install

```bash
git clone https://github.com/yourusername/server-configs.git ~/.personal_config
~/.personal_config/install.sh
```

## Usage

- Activate: `myenv`
- Deactivate: `deactivate`
